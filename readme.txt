Download zip file of project from github and open in your workspace. 
The driver file is located in the tests package. Run the driver.
You may change the capacity of the boat to see the functionality of the program, without having to add as many reservations.
If you put the capacity at 0, you will not be able to make a reservation.
The program will prompt user for haw many reservations they would like to make.
This value is used to loop and ask user for reservation input.
If the capacity is enough to accommodate all reservations, the program displays the manifest, or the results of all reservations.
If the capacity is not enough to accommodate all reservations, the program informs user and displays the accepted reservations.
